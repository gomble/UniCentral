Agent-Enrollment

Mit diesem Key können sich Maschinen selbst registrieren. Die Verbindung wird nach Registrierung per HMAC-Signatur gesichert.

{{ settingsForm.enrollmentKey }}
powershell -Command "iwr {{ baseUrl }}/api/agent/install-script/windows -UseBasicParsing | iex" {{ settingsForm.enrollmentKey }} server
curl -sL {{ baseUrl }}/api/agent/install-script/linux | bash -s -- {{ settingsForm.enrollmentKey }} server

Ersetze "server" mit "client" für Client-Maschinen.
